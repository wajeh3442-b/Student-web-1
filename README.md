# HTML Links Assignment

## 📌 Description

This project is a simple HTML webpage that demonstrates the use of hyperlinks. It includes links to popular search engines and well-known websites. Each link opens in a new browser tab for better user experience.

## 🛠 Features

* Organized layout using containers
* Two sections:

  * Search Engine Links
  * Popular Website Links
* Links open in a new tab using `target="_blank"`
* Styled using internal CSS
* Clean and user-friendly design

## 💻 Technologies Used

* HTML5
* CSS3

## 📂 Structure

* `index.html` – Main file containing all code

## 🚀 How to Run

1. Download or copy the HTML file
2. Open it in any web browser
3. Click on links to visit websites

## 🎯 Purpose

This assignment helps beginners understand:

* How to create hyperlinks in HTML
* How to structure a webpage
* Basic styling using CSS
